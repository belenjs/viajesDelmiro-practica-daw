# Viajes Delmiro

Landing page desarrollada como práctica de HTML y CSS.

## Descripción

Página web con temática fija: una agencia de viajes ficticia cuya propuesta es:

**Los adultos viajan gratis y los niños pagan.**

El objetivo es crear una web sencilla, visual y bien estructurada.

## Contenido de la página

* Cabecera con navegación
* Sección principal (hero)
* Destinos o promociones
* Información sobre la oferta
* Formulario de contacto
* Pie de página

## Tecnologías

* HTML5
* CSS3

## Características

* Uso de HTML semántico
* Estilos en CSS externo
* Uso de Flexbox
* Uso de CSS Grid
* Diseño responsive básico
